دليل الصفحة

هذه الصفحة تتيح لك تخزين الوثائق الأساسية للشركة بأنواعها المختلفة، مع ميزة التنبيه التلقائي عند اقتراب هذه الصفحة تحتوي على نموذج لملف اليومية الذي يمكنك تنزيله من خلال زر "Download"، حيث ستجد القيود اليومية الخاصة بفواتير المبيعات والمشتريات جاهزة داخل الملف. كل ما عليك فعله هو إضافة القيود اليومية التي لا تتوفر بياناتها على موقع الفاتورة الإلكترونية، ثم إعادة رفع الملف من خلال زر "Upload". عندها سيتم حذف الملف القديم واستبداله بالجديد للحصول على قوائم الدخل المحدثة.

Page Guide

This page contains a template for the journal file, which you can download using the "Download" button. Inside the file, you'll find the journal entries for sales and purchase invoices already prepared. All you need to do is add the journal entries that aren't available on the Electronic Invoice Portal, then re-upload the file using the "Upload" button. The old file will then be deleted and replaced with the new one to obtain updated financial statements.
